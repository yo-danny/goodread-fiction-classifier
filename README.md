## Classifying Books from Their Blurbs

This project demonstrates how to use a recurrent neural network (RNN) to classify books as either fiction or non-fiction based on their blurbs or descriptions.

It follows a complete data science workflow, covering everything from data extraction to model deployment.

This work was inspired by Naren Santhanam’s article on Medium:
[Full Stack Data Science Project – Part 1](https://medium.com/data-science/a-full-stack-data-science-project-part-1-9f73a997dc4d)
